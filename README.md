# Информационный портал кассира

Этот репозиторий содержит статические HTML-страницы:

- `index.html` — интерфейс кассира (просмотр/поиск).
- `admin.html` — интерфейс контент-менеджера (добавление/удаление, демо-режим).

## Быстрый просмотр локально

Откройте файлы двойным кликом в проводнике/файловом менеджере:

- `index.html`
- `admin.html`

## Публикация через GitHub Pages

1. Откройте репозиторий на GitHub.
2. Перейдите **Settings → Pages**.
3. В **Source** выберите **Deploy from a branch**.
4. В **Branch** выберите вашу ветку (обычно `main`) и папку `/ (root)`.
5. Нажмите **Save** и дождитесь публикации.

После публикации страницы будут доступны по адресу:

- `https://<ваш-логин>.github.io/<имя-репозитория>/index.html`
- `https://<ваш-логин>.github.io/<имя-репозитория>/admin.html`

Если вы видите старую страницу, проверьте:

- что GitHub Pages публикует **ту же ветку**, где находятся эти файлы;
- что публикация выполнена из **корня репозитория**, а не из другой папки;
- что страница уже пересобралась (иногда требуется 1–2 минуты).

## Хранение данных без сервера

Прототип работает **без сервера** и сохраняет записи в **localStorage браузера**:

- добавления/удаления в `admin.html` сохраняются локально;
- `index.html` подхватывает эти данные автоматически;
- данные видны только на этом компьютере и в этом браузере.

Чтобы перенести данные между браузерами/ПК:

1. В `admin.html` нажмите **Экспортировать** и скопируйте JSON.
2. На другом компьютере вставьте JSON в поле и нажмите **Импортировать**.
